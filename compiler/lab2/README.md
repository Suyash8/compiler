# lab/lab2 — Regex examples and simple lexer

This folder contains Python scripts implementing the examples and exercises transcribed from `CSE304_Lab1_Compiler_Design.pdf`.

Files added:

- `example1_search.py` — demo of `re.search()`
- `example2_match_vs_search.py` — shows `re.match()` vs `re.search()`
- `example3_findall.py` — `re.findall()` example
- `example4_sub.py` — `re.sub()` example
- `example5_split.py` — `re.split()` example
- `validators.py` — password, email, and URL validators with test cases
- `lexer.py` — simple regex-based lexer demonstrating tokenization
- `run_all.sh` — run all example scripts

Run examples:

```bash
python3 lab/lab2/example1_search.py
python3 lab/lab2/example2_match_vs_search.py
python3 lab/lab2/example3_findall.py
python3 lab/lab2/example4_sub.py
python3 lab/lab2/example5_split.py
python3 lab/lab2/validators.py
python3 lab/lab2/lexer.py
```
